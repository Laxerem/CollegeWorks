# SQLite

> Домашнее задание по SQLite.

## Обзор

В этом проекте представлены базовые возможности работы с SQLite:
* Добавление групп, студентов в базу данных
* Удаление студентов из базы данных
* Получение студента по ID
* Получение списка студентов по id группы
* Получение списка студентов с определённым шифром специальности

## Структура проекта

```
.
├── README.md # Документация
└── src # Исходный код
    ├── database
```

## Модель данных

### Диаграмма сущностей и связей

```
┌─────────────────────────────────┐
│          Group                  │
├─────────────────────────────────┤
│ id: int                         │
│ name: str                       │
│ year: int                       │
│ speciality_code: str            │
└─────────────────────────────────┘
          ▲
          │ 1
          │
          │ (group_id)
          │
          │ 0..*
┌─────────────────────────────────┐
│         Student                 │
├─────────────────────────────────┤
│ id: int                         │
│ name: str                       │
│ birthday: date                  │
│ group_id: int (FK)  ◄──────────┘
└─────────────────────────────────┘
```

**Связь:** Студент принадлежит к группе через внешний ключ `group_id`

## Как запустить

1. Установить [python](https://www.python.org/downloads/)
2. Запустить программу:
   ```bash
   cd src
   python main.py
   ```

Программа создаст базу данных SQLite, добавит тестовые данные (группы и студентов) и выведет результаты операций в консоль.
